# Lojas IFTO

Projeto desenvolvido para a disciplina de Programação Web Design.

## Objetivo

Desenvolver um sistema web para gerenciamento de produtos utilizando Flask, SQLite e Bootstrap.

## Tecnologias Utilizadas

* Python
* Flask
* SQLite
* HTML
* CSS
* Bootstrap

## Funcionalidades

* Cadastro de produtos
* Listagem de produtos
* Edição de produtos
* Exclusão de produtos
* Dashboard com indicadores

## Como Executar

1. Instale as dependências:

pip install -r requirements.txt

2. Execute o sistema:

python app.py

3. Abra o navegador:

http://127.0.0.1:5000
